    Compilation
    ===========
    
    make        -> Emit the binary ./mcc in the root directory. 
    make clean  -> Remove the binary and all compilation artifacts.

    Usage
    =====

    ./mcc file 

    Emits the stack code for the given Minisculus source to stdout. 
    Displays an error message if the source code is invalid, including
    lexical and grammatical errors.

    Running Code
    ============

    Refer to the stack.csh stack machine implementation provided by Dr.
    Cockett to execute the emitted stack machine code.
